Affiliation Decitre

== Description ==

This plug-in is designed to really simply manage your Decitre affiliation links.

It provides short-codes to create links or buttons to the product ordering pages based on the Decitre product ID. For marketing purposes, it allows adding Google Analytics or Google Tag Manager based tracking.

This plug-in is not provided by Decitre.

== Installation ==

1. Upload the plugin files to the <code>/wp-content/plugins/affiliation-decitre</code> directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the "Plugins" screen in WordPress.
1. Use the Settings -> Decitre Affiliation screen to configure the plugin.
1. Enter your Decitre Affiliation credentials.

== Frequently Asked Questions ==
